# Design system

EduRecall uses a light-green learning landscape with dark-forest code areas, warm gamification accents and a calmer teacher workspace. It takes only high-level inspiration from mission maps and learning dashboards; no logo, mascot, asset, copy or pixel-perfect layout is copied.

## Tokens

Tokens live in CSS and `packages/design-system/tokens.json`. Primary `#4AAA64`, forest `#173622`, background `#F4FAF5`, yellow `#FFD66E`, blue `#72B9F2`, purple `#AA8CE9`, orange `#FF9F66`, danger `#DF5E63`.

## Accessibility

- semantic buttons and links;
- visible keyboard focus;
- skip link;
- contrast-aware text and states;
- progress ARIA values;
- keyboard sensor for Code Order;
- `prefers-reduced-motion` disables nonessential animation;
- responsive student bottom navigation;
- loading, empty, offline and error visuals.

The `/design-system` page shows foundations and components; `/design-system/assets` renders the complete manifest.
